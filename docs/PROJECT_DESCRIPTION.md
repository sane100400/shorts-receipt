# 쇼츠 영수증 (Scroll Receipt)

> **Track: Gemini in Entertainment**
> 오늘의 스크롤, 영수증 나왔습니다. — Your daily scroll, receipted.

## 한 문장 소개

쇼츠 영수증은 오늘 PC Chrome에서 본 YouTube Shorts의 개수·활성 시청 시간·1초 컷·완주·반복을 자동으로 기록하고, **Gemini가 오래 본 대표 쇼츠의 장르와 편집 문법을 분석**해 친구에게 자랑할 수 있는 한 장짜리 일일 쇼츠 전적표(9:16 공유 카드)로 만들어 주는 Chrome 확장 프로그램입니다.

## 왜 만들었나 — 쇼츠 시청량은 자랑거리다

한국의 10대 후반~20대에게 쇼츠 시청량은 숨겨야 할 숫자가 아니라 친구 사이의 농담이자 경쟁 소재입니다. "나 어제 쇼츠 200개 봄"은 이미 대화의 일부죠. 하지만 지금 보여줄 수 있는 건 스크린 타임 캡처나 시청 기록 목록뿐 — 오늘 **몇 개**를 봤는지, **어떤 장르**에 시간을 썼는지, 얼마나 빨리 넘겼는지는 아무도 보여줄 수 없습니다.

Spotify Wrapped가 음악 취향을 정체성과 공유 콘텐츠로 바꾼 것처럼, 쇼츠 영수증은 하루의 쇼츠 소비를 **오늘 바로 친구에게 보낼 수 있는 엔터테인먼트 결과물**로 바꿉니다.

> 실제 시청 행동 + Gemini의 콘텐츠 이해 + 숫자가 크게 보이는 공유 카드 = 오늘의 쇼츠 영수증

## 무엇을 보여주나

- **오늘 본 쇼츠 143개 · 총 활성 시청 46분 18초** — 백그라운드·일시정지 시간을 제외한 진짜 시청 시간
- **1초 컷 61회 · 완주 24회 · 반복 3회** — 넘긴 방식까지 기록되는 시청 습관
- **오늘의 최애 장르: 썰·사연 42%** — Gemini가 분석한 장르별 시간 비중
- **오늘 만난 쇼츠 문법** — 큰 중앙 자막, TTS, Part 2 클리프행어 같은 편집 문법 통계
- **오늘의 칭호와 배지** — "백쇼츠 클럽", "번개 손절" 같은 게임 전적표식 라벨
- **"오늘의 주인공은 고양이가 아니라 남의 퇴사였습니다."** — Gemini가 검증된 사실만으로 만드는 펀치라인
- 이 모든 것을 **1080×1920 PNG 공유 카드**로 저장 — 카드 하단의 "너 오늘 몇 개 봄?"이 친구의 설치를 유도합니다

## Gemini를 어떻게 썼나

Gemini가 없으면 이 제품은 시청 타이머에 불과합니다. Gemini가 있어야 "143개를 봤다"에서 "그중 시간을 가장 많이 쓴 건 아이돌 직캠과 TTS 썰이었다"로 넘어갑니다.

- **멀티모달 콘텐츠 이해**: 시청 시간이 가장 긴 대표 쇼츠 5개 + 가장 빨리 스킵한 대조 영상 1개의 공개 URL을 Gemini(URL context)에 전달해 주 장르·포맷·첫 훅 유형·payoff 시점·편집 문법(TTS, 빠른 컷, 루프 편집 등)을 분석합니다.
- **구조화 출력(Structured Output)**: 고정 JSON Schema로 응답을 강제하고, 각 판정에 타임스탬프 근거와 confidence를 요구해 스키마 검증을 100% 통과한 결과만 사용합니다.
- **엄격한 역할 분리**: 모든 숫자·비율·칭호·배지는 결정적 코드가 계산하고, Gemini는 콘텐츠 이해와 검증된 fact 기반의 45자 펀치라인만 담당합니다. AI가 숫자를 지어낼 수 없는 구조입니다.
- **우아한 강등(Graceful Degradation)**: URL context 불가 시 metadata 분석으로, 전체 실패 시 행동 전용 영수증으로 강등되어 데모가 절대 멈추지 않습니다.

## 기술 구성

| 구성 요소 | 기술 |
|---|---|
| 클라이언트 | Chrome Extension (Manifest V3) — SPA 네비게이션 추적, 활성 영상 판정, 500ms tick 기반 활성 시청 측정 |
| 저장소 | `chrome.storage.local` — 정산 전 모든 원시 기록은 로컬에만 보관 |
| 백엔드 | Node.js 22 on **Cloud Run** (asia-northeast3) |
| AI | **Gemini 3.5 Flash** — URL context + 구조화 JSON 출력, 영상별 실패 격리, 40초 hard deadline |
| 공유 | Canvas 기반 1080×1920 PNG 영수증 카드 생성 |

## 프라이버시 원칙

- 로그인 정보·쿠키·댓글·검색 기록·다른 웹사이트는 일절 수집하지 않습니다.
- 사용자가 명시적으로 "정산"을 누를 때만, 대표 영상 최대 6개의 URL·제목·행동 통계와 일일 집계만 서버로 전송합니다. 나머지 수백 개 영상의 기록은 브라우저를 떠나지 않습니다.
- 공유 카드에는 개별 영상 제목·채널·URL을 기본적으로 포함하지 않습니다.
- 중독·치료 같은 훈계 표현과 민감한 개인 특성 추론을 금지합니다. 이건 자랑을 위한 전적표지, 진단서가 아닙니다.

## 데모 하이라이트

1. Shorts 6개를 실제로 넘기면 팝업 숫자가 실시간으로 올라갑니다. (자동 행동 추적)
2. "오늘 쇼츠 영수증 보기" → 45초 안에 Gemini 분석이 결합된 일일 영수증이 생성됩니다.
3. 영수증 종이 질감의 9:16 공유 카드를 PNG로 저장해 바로 친구에게 보냅니다.

**YouTube는 무엇을 추천했는지 알지만, 쇼츠 영수증은 오늘 내가 무엇에 시간을 썼는지를 친구에게 보여줍니다.**
